# SimplStream

A modern streaming app for Android TV, built with Kotlin and Jetpack components.

![Android](https://img.shields.io/badge/Platform-Android%20TV-green)
![Kotlin](https://img.shields.io/badge/Language-Kotlin-purple)
![API](https://img.shields.io/badge/Min%20SDK-21-blue)

## Features

- 🎬 Browse movies and TV shows with a beautiful Leanback UI
- 🔍 Search across your media library
- 📺 Optimized for Android TV with D-pad navigation
- 🎯 Multiple streaming servers for reliability
- ⭐ Clean, modern Material Design interface

## Tech Stack

- **Language:** Kotlin
- **Architecture:** MVVM + Clean Architecture
- **DI:** Hilt
- **Networking:** Retrofit + OkHttp
- **Image Loading:** Coil
- **Navigation:** Jetpack Navigation Component
- **UI:** Android TV Leanback Library

## Building

1. Clone the repository
```bash
git clone https://github.com/yourusername/SimplStream.git
```

2. Open in Android Studio

3. Build and run on an Android TV device or emulator

## Requirements

- Android Studio Arctic Fox or newer
- Android SDK 21+
- Android TV device or emulator
